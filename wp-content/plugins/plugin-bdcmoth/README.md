# Plugin BDC-Moth

BDC Moth theme utility plugin.

### Description

The BDC-Moth plugin handles two custom post types and their custom taxonomies.

### Installation

1. Upload.
2. Activate.

### Details

The Focus Areas taxonomy is shared across all post types.

#### Post Types

* Team
* Tool

#### Custom Taxonomies

* Focus Areas
* Team Types
* Tool Types
